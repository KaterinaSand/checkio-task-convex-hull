Вам дан список точек на прямоугольной координатной плоскости. Нужно найти выпуклую облочку образуемую этими точками. Выпуклая оболочка для множества X точек - это наименьшее выпуклое множество, которое содержит X. Для наглядности,если X - это множество гвоздиков вбитых в плоскость, то выпуклую оболочку можно представить, как резиновую ленту обтянутую вокруг этих "гвоздиков". Если точка лежит на грани фигуры, то считаем ее включенной в выпуклую оболочку.

Точки представлены, как последовательность координат в виде [x, y], где x и y - целые числа. Результат должен быть представлен, как последовательность индексов точек из изначальной последовательности. Точки описываются в порядке по часовой стрелке и последовательность должна начинаться с нижней самой левой точки.

Входные данные: Список (list) координат, где каждая координата есть список двух целых чисел (int).

Выходные данные: Последовательность точек, как список (list) целых чисел (int).

Примеры:

convexHull([[7, 6], [8, 4], [7, 2], [3, 2], [1, 6], [1, 8], [4, 9]]) == [4, 5, 6, 0, 1, 2, 3]
convexHull([[3, 8], [1, 6], [6, 2], [7, 6], [5, 5], [8, 4], [6, 8]]) == [1, 0, 6, 3, 5, 2]
{% else %} 
convex_hull([[7, 6], [8, 4], [7, 2], [3, 2], [1, 6], [1, 8], [4, 9]]) == [4, 5, 6, 0, 1, 2, 3]
convex_hull([[3, 8], [1, 6], [6, 2], [7, 6], [5, 5], [8, 4], [6, 8]]) == [1, 0, 6, 3, 5, 2]
{% endif %}

Зачем это нужно: Выпуклая оболочка имеет практическое применение в системах распознавания образов, обработке изображений, статистики и других областях. Также это очень удобно использовать в различных системах для строительства и архитектуры.

Precondition:
2 < len(coordinates) < 10
all(0 < x < 10 and 0 < x < 10 for x, y in coordinates)
